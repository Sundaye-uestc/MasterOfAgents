## 一、PPT 生成端到端验证

### 1.1 基础 PPT 生成

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 新建单聊对话 | 对话创建成功 |
| 2 | 发送："帮我生成一个关于人工智能发展的5页PPT" | Agent 开始执行 |
| 3 | 等待 Agent 完成 | 回复中提及生成的文件 |
| 4 | 查看消息下方 InlineArtifactCard | 出现 PPT 下载卡片或轮播卡片 |
| 5 | 打开工作区 → 文件 Tab | 文件树中能看到 .pptx 或 slides_plan.json |
| 6 | 打开工作区 → 快照 Tab | 快照标签为 "对话前/后 · {对话标题} · "{消息预览}""

### 1.2 ImageSlideshowCard 轮播

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 若 Agent 生成了 slideshow 类型 artifact | 消息下方出现图片轮播组件 |
| 2 | 点击轮播右侧区域 | 翻到下一页 |
| 3 | 点击轮播左侧区域 | 翻到上一页 |
| 4 | 按键盘 ← 键 | 翻到上一页 |
| 5 | 按键盘 → 键 | 翻到下一页 |
| 6 | 查看页码指示器 | 显示 "2/5" 格式 |

### 1.3 PPTX 下载

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 若 Agent 生成 .pptx 文件 | 出现 PPT 下载卡片（橙色渐变色） |
| 2 | 点击下载按钮 | 浏览器下载 .pptx 文件 |

## 二、快照系统改进验证

### 2.1 有意义标签

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 进行至少 1 轮对话（让 Agent 创建/修改文件） | Agent 正常完成 |
| 2 | 打开右侧工作区 → 快照 Tab | 快照列表不为空 |
| 3 | 查看快照标签 | 格式为 "对话前 · {标题} · "{消息前20字}"" 或 "对话后 · ..." |
| 4 | 不同对话的快照 | 标签能明确区分不同对话 |

### 2.2 展开查看文件变更

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 点击一个快照条目 | 快照展开，下方显示关联的文件变更列表 |
| 2 | 查看变更条目 | 显示：变更符号(+/−/~) + 文件路径 + 改动行数 |
| 3 | 查看 diff 片段 | 展开区域底部有 diff 代码预览，+ 绿色 / - 红色 / @@ 蓝色 |
| 4 | 再次点击同一个快照 | 折叠收起 |
| 5 | 点击另一个快照 | 切换到新快照展开 |
| 6 | 无关联变更的快照 | 显示 "此快照没有关联的文件变更" |

### 2.3 回滚 Toast 反馈

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 在快照条目上 hover → 点击 "↩ 回滚" | 出现确认提示 "确认回滚到此快照？将覆盖当前所有文件。" |
| 2 | 点击 "确认回滚" | 右下角弹出绿色 Toast "✅ 已回滚到 xxx"，3 秒后消失 |
| 3 | 查看文件树 | 文件自动刷新 |
| 4 | 点击 "取消" | 确认提示消失，不执行回滚 |

## 四、富媒体内联展示回归

### 4.1 InlineDiffCard

| 步骤 | 操作 | 预期结果 |
|---|---|---|
| 1 | 让 Agent 创建一个新文件 "test.txt"，内容任意 | Agent 创建文件 |

### 4.2 WebPreviewCard（HTML 预览）

| 步骤 | 操作 | 预期结果 |
